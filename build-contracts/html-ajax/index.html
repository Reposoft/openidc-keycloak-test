Initial setup

You need to import a testing realm into Keycloak. Login at http://openidc:8080/ (URL depends on your hosts file) using admin:openidctest. In create realm use the file Testrealm-realm.json from keycloak-setup/exports. Then import Testrealm-users-0.json as Partial import.

Auth testing

Go to /protected to log in.

The user is test1:test1 user, if Keycloak realm import worked.
